# Debos---Toolbox
The ultimate all-in-one GUI toolbox for Debian and LMDE. Effortlessly clean your system, manage custom kernels, install essential software, configure hardware drivers, and customize your desktop experience—all from a single, elegant interface.
The Easy Way (GUI - Just Click!)

Open your File Manager and navigate to your Downloads folder.

Simply double-click the debos-toolbox_0.4.29.1_all.deb file.

Your default software installer (such as GNOME Software, GDebi, or Discover) will open.

Click Install, enter your administrator password, and wait for the process to finish.

From Terminal

cd ~/Downloads

sudo apt update

sudo apt install ./debos-toolbox_0.4.29.1_all.deb
